# Task List

A simple task list.

It's my study for MVC.

## Setup

Requirements:

* [Node.js](http://nodejs.org/) (~> 0.6.11)
* [CoffeeScript](http://jashkenas.github.com/coffee-script/) (~> 1.2.0)

``` sh
$ git clone git://github.com/tricknotes/task-list.git
$ cd task-list
$ npm install .
```

And run:

``` sh
coffee app.coffee
```

## Test

Open your browser `test/index.html`
