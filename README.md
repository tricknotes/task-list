# Task List

A simple task list.

It's my study for MVC.

## Setup

Requirements:

* [Node.js](http://nodejs.org/) (~> 0.6.11)

``` sh
$ git clone git://github.com/tricknotes/task-list.git
$ cd task-list
$ npm install .
```

And run:

``` sh
npm start
```

If you want to listen other port:

``` sh
PORT=3000 npm start
```

## Test

Open your browser `test/index.html`
